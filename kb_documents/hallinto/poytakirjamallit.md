| Käytä | Älä käytä |

| "Hallitus päätti..." | "Keskusteltiin..." |
| "Todettiin..." | "Mietittiin..." |
| "Hyväksyttiin..." | "Ajateltiin..." |
| Passiivimuoto | Aktiivimuoto |
